Call Stack in JS

Call Stack is a data structure to manage running code, paused code in JS because JS is Single Threaded Lang.

Call Stack follow the principle: LIFO :: Last-In First-Out

How Call Stack executes JS code ?

Its takes code and execute and then remove all the code from call stack.

But, in case of function it take the function create the lexcal scope and then execute then after ended it delete this.

when maximum call stack size exceeded that we call as Stack Overflow. Means Overflow of the Call Stack
