Event Delegation in JavaScript

What is Event Delegation?

Event Delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, we add a single event listener to their parent element. The parent listens for events that bubble up from its children and handles them accordingly using the event object.

Why use Event Delegation?

  • Improves performance by reducing the number of event listeners
  • Follows the DRY (Don't Repeat Yourself) principle
  • Works automatically for dynamically added elements

How does it work?

When an event occurs on a child element, it "bubbles up" to its parent. We can detect which child triggered the event using event.target and apply logic based on that.
